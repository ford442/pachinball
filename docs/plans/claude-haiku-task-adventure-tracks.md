# Claude Haiku Task — Adventure Mode: New Tracks & Goals (Pachinball)

**Repo**: https://github.com/ford442/pachinball  
**Focus**: Expand Adventure Mode with new tracks and goal systems.

## Goal
Add **new adventure tracks** and **meaningful goals** to make Adventure Mode deeper and more replayable.

## Features to Implement

### 1. New Adventure Tracks
- Create 2–3 new distinct adventure tracks with unique layouts/themes
- Different visual styles and difficulty curves
- New background elements, lighting, and music triggers

### 2. Goal System
- Add clear, track-specific goals (e.g., "Hit all spinners", "Collect 5 gold balls", "Survive 60 seconds")
- Progress tracking and completion rewards
- Multiple goal types: score-based, collection-based, survival, combo-based

### 3. Track Selection & Progression
- Improve track selection screen with previews
- Unlock system (new tracks unlock after completing previous ones)
- Save/load progress for completed tracks and goals

### 4. Polish
- Better camera transitions between tracks
- Clear goal UI and feedback
- Nice completion animations and rewards

## Instructions
- Work in the existing AdventureManager and track system
- Keep goals clear and satisfying to complete
- Make tracks feel distinct and fun
- Integrate with EventBus for lighting, sound, and effects

Start by adding 1–2 new tracks with clear goals. Make Adventure Mode feel much richer. Go!
